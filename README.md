# SafeRoadIndia Analytics Dashboard

An interactive Streamlit dashboard for analyzing Indian Road Accident data (2022–2025).

## Features

- Interactive Filters
- Accident Hotspot Map
- State & City Analysis
- Severity Analysis
- Cause Analysis
- Time Analysis
- Machine Learning Risk Predictor
- AI-powered Risk Explanation

## Technologies

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Scikit-Learn
- Folium

## Dataset

Indian Road Accident Dataset (Kaggle)
